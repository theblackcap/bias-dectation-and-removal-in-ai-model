Bringing Fairness to Machine Learning Models.

An end-to-end framework to detect predictive demographic biases in hiring algorithms, and cleanly mitigate them via Fairness through Unawareness.

An initial Logistic Regression classification model is built utilizing your tabular hiring data. The model takes in contextual features alongside sensitive demographic data (like Gender).

We measure disparity using industry-standard definitions:
  • Statistical Parity Diff: Absolute selection differences. (Ideal: 0)
  • Disparate Impact: "80% Legal Rule" thresholds of group success proportional ratios. (Ideal: 0.8 - 1.25)

Following detection, an active data scrubbing step eliminates the direct sensitive features entirely, retraining a Mitigated Framework to compare statistical performance vs. core equity gains side-by-side visually!
